Project Name : AI Chatbot

Explanation : This Android-based chatbot application for pharmacies is built using Flutter and Dart for a responsive user interface (UI) on the Android platform. The chatbot in this application allows users to interact and get drug recommendations based on their symptoms or health conditions. On the backend, the application uses Python and Flask to handle server-side logic, such as processing user symptom data and searching for drugs in the pharmacy database. In addition, this application utilizes the REST API to connect the frontend (Flutter application on Android) with the backend (Flask) which allows data communication between the mobile application and the server.

Build with : Flutter, Dart, Python, and Flask

Year : 2024
